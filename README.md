# BIG-DRIFTER
![DRIFTER](DRIFTER.png)
BIG DRIFTER IS WATCHING YOU

## Information
Find Gambit/Gambit Prime AFK user from clan member list.

## How to use
1. `API_KEY`에 번지 API KEY, `TELEGRAM_TOKEN`에 텔레그램 봇 토큰.
2. 주기적으로 `main.py`를 cron 돌리시면 됩니다. 권장 시간은 5분~10분 사이.
